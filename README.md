# Temple Billing System (MERN)

## Project Structure
- `backend` - Node.js + Express + MongoDB API
- `frontend` - React + Vite app

## Backend Setup
1. Go to `backend`
2. Copy `.env.example` to `.env`
3. Update `MONGODB_URI`
4. Run `npm run dev`

## Frontend Setup
1. Go to `frontend`
2. Run `npm run dev`

Frontend runs on `http://localhost:5173` and proxies API calls to backend `http://localhost:5000`.
